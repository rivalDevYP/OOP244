Reflection Questions

1) Does the Hero class need to know about the existence of the SuperHero class?
	No, the base class need not know the derived classes existence. It's purpose is to provide the methods/variables for derived classes to inherit.
	
2) Does the SuperHero class need to know about the existence of the Hero class?
	Yes, the derived class must know that the base classes exist. This can be done by including the name of the header file that contains the class of which you are attempting to inherit from.
